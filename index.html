Proyect THEORY:

Idea general:

Esta página es el proyecto THEORY(no es muy original el nombre pero sirve). Acá voy a intentar subir y actualizar todos los contenidos teóricos vistos en Aprende Programando 2, con el fin de al terminar el cuatrimestre, tener una especie de resumen donde sea más facil buscar el contenido que se necesite reforzar o simplemente recordar relacionado con JavaScript ya que ese es el lenguaje que estamos aprendiendo a usar en este nivel.

START

Contenidos:

  • Class 1 (Introducción a Javascript)
  • Class 2 [Variables (let y const). Operadores de asignación, incremento y decremento (i++, i--). Operaciones aritméticas desde la consola [parseInt(), parseFloat()]. Concatenación de variables.]
  • Class 3 (Estructura IF-ELSE, If Else Fallback, Operadores Lógicos y de Comparación)
  • Class 4 [Funciones, qué son y para qué se utilizan, Tipos de funciones, Funciones anónimas, Funciones con parámetros, Return, Callbacks, Scope (alcance de variables)]
  • Class 5 (Arrays, Introducción a Bucles For y bucles)
  • Class 6 (Bucles || y While and Do While)
  • Class 7 [Arrays y Objetos (como se relacionan entre sí). Objeto Math. Métodos en objetos]
  • Class 8 (DOM, querySelector)
  • Class 9 (DOM e implementación de atributos de manera dinámica)
  • Class 10 (Eventos. Target. Capturar eventos desde el teclado)
