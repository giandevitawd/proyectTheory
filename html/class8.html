El DOM:

El DOM (Document Object Model) es una estructura jerárquica que representa un documento HTML o XML en una página web. Básicamente, el DOM es una representación del contenido de una página web en forma de árbol de nodos, donde cada elemento HTML se convierte en un objetodel DOM.
El DOM es muy importante porque permite a las y los desarrolladores web interactuar y manipular los elementos HTML de una página web utilizando JavaScript. Es decir, a través del DOM, podemos acceder y modificar el contenido, atributos y estilos de los elementos de una página web.

Desde el DOM al HTML...

Ahora que ya sabemos como acceder al DOM, determinamos cómo se ven sus ramas por dentro con console.dir(document); , el siguiente paso será elegir un elemento de este árbol de elementos que representa a todo el archivo HTML de la página web para que podamos manipularlo desde Javascript. Para seleccionar elementos del HTML desde Javascript tenemos varios métodos. Podemos elegirlos por su id, por su class o su nombre (li, div, section, h2, etc). Existen varios métodos para realizar esta selección.
getElementBy() es un método en JavaScript que se utiliza para seleccionar elementos HTML dentro de un documento. Este método toma como parámetro el nombre de la etiqueta HTML o el valor del atributo "id" que deseas seleccionar y devuelve un objeto que representa el elemento HTML seleccionado.

querySelector:

querySelector() es un método en JavaScript que nos permite seleccionar elementos de una página web utilizando CSS selectors. Este método podemos utilizarlo para seleccionar elementos del HTML y guardarlos en una constante para que esté siempre disponible en JavaScript. Una vez que tenemos seleccionado el elemento que nos interesa utilizando querySelector, podremos modificar sus características de CSS pero desde JavaScript. Para esto se utiliza la propiedad style.

Repasando más propiedades con querySelector...

La función querySelector() en JavaScript, como mencionamos en el tema anterior, permite seleccionar un elemento HTML utilizando selectores y luego acceder a sus propiedades a través del objeto resultante. Revisemos su estructura:

Además de modificar características y propiedades visuales de los elementos con CSS y desde Javascript, también podemos crear elementos, sumarlos al HTML, e incluso eliminarlos, modificar sus clases asignadas, cambiar su contenido y mucho más. Esto es posible gracias a propiedades que se habilitan en Javascript con el uso de querySelector y dichas propiedades se aplican luego en aquellos elementos que estamos seleccionando. En el tema anterior conocimos muchas de ellas, pero acá te compartimos las propiedades más comunes y ejercicios prácticos con su implementación:

  • classList: Es una propiedad que permite manipular las clases CSS de un elemento de manera sencilla, proporcionando métodos para agregar, eliminar y verificar clases en un elemento HTML. Algunos métodos comunes son `add` para agregar una clase, `remove` para eliminar una clase, `toggle` para alternar una clase y `contains` para verificar si un elemento tiene una clase específica.
  • textContent: Es una propiedad que se utiliza para acceder al contenido de texto de un elemento HTML y modificarlo. Al acceder a textContent, obtendremos una representación de texto plano de todos los contenidos de texto dentro del elemento, incluyendo cualquier espacio en blanco o saltos de línea. La diferencia con innerHTML es que este últimos nos devolverá tanto el contenido de texto como las etiquetas HTML dentro del elemento. Por ejemplo, si tienes un elemento div con contenido HTML dentro, al acceder a innerHTML obtendrás todo el contenido, incluyendo las etiquetas y su contenido interno. Por otro lado, la propiedad textContent devuelve o establece el contenido de texto sin formato de un elemento, excluyendo cualquier etiqueta HTML dentro del elemento. Si accedes a textContent, obtendrás solo el contenido de texto plano sin ninguna estructura HTML. Esto es útil cuando solo necesitas trabajar con el texto en sí y no te importan las etiquetas HTML.
  • innerHTML:Es una propiedad que representa el contenido HTML dentro de un elemento. Permite acceder y modificar el contenido HTML interno de un elemento, incluyendo etiquetas, texto y otros elementos HTML anidados. Al asignar un valor a `innerHTML`, se reemplaza todo el contenido anterior del elemento. Por ejemplo, el contenido de un elemento p es un texto. Lo que podemos hacer con esta propiedad es modificar el texto que contiene ese elemento desde Javascript.
  • setAttribute: Es un método que permite establecer un atributo en un elemento HTML. Toma dos argumentos: el nombre del atributo y su valor. Puede utilizarse para establecer o modificar cualquier atributo en un elemento, como `src`, `href`, `class`, etc.
  • getAttribute: Es un método que permite obtener el valor de un atributo específico de un elemento HTML. Toma un argumento que es el nombre del atributo que se desea obtener. Por ejemplo: elemento.getAttribute('src')` devolverá el valor del atributo "src" del elemento.
  • appendChild: Método que se utiliza en JavaScript para agregar un nuevo elemento (nodo) como hijo de otro elemento existente en el DOM (Documento de Objetos del Modelo). Lo importante a tener en cuenta con este método es que el nuevo elemento que se agregue se suma al final de una lista de elementos contenidos en un “padre".

querySelectorAll:

Como habrás notado en temas anteriores, getElementBy() solo selecciona el primer elemento HTML que coincide con el parámetro de búsqueda. Si deseas seleccionar todos los elementos que coinciden con la búsqueda, debes utilizar querySelectorAll().Lo que devuelve este último método se lo conoce como NodeList. Un NodeList es un objeto que representa una colección de elementos HTML en el orden en que aparecen en el documento. Es similar a un array en su sintaxis, pero no tiene todos los métodos de un array, como el método "push" o "pop".
querySelectorAll() lo utilizamos entonces para seleccionar varios elementos del DOM que cumplen con ciertos criterios y, a continuación, manipularlos mediante programación. Además de poder modificarlos, al devolvernos algo similar a un array nos permite recorrerlo con un for. Esto puede ser útil, por ejemplo, para cambiar el estilo de varios elementos al mismo tiempo, para agregar o quitar clases a un grupo de elementos, o para agregar event listeners a múltiples elementos de manera eficiente.
