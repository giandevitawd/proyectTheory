Variables y constantes:

Todos los lenguajes de programación tienen variables. Para crear una variable tenemos que usar la palabra let, esta tiene que tener un nombre y es un elemento que sirve para guardar un valor.
En el archivo JS se escribe de la siguiente manera...
let nombredelavariable = "valordelavariable";
A diferencia de las variables, se pueden crear constantes. Estas tienen la particularidad de que no pueden cambiar de valor a lo largo del programa, a diferencia de las variables que sí pueden cambiar su valor.
En el archivo JS se escribe de la siguiente manera...
const nombredelaconstante = "valordelaconstante";

Operaciones aritméticas:

  • Suma: La operación suma se realiza concatenando los números que queremos sumar con el signo de la suma (+).
  • Resta: De manera análoga, la operación resta se realiza concatenando los números que queremos restar con el signo de la resta (-).
  • Multiplicación: La operación multiplicación se realiza concatenando los números que queremos multiplicar con el signo de la multiplicación, el asterisco (*).
  • División: La operación división se realiza ubicando signo de la división, la barra inclinada hacia la derecha (/), entre los números que queremos dividir.

    • Además de las operaciones aritméticas básicas, existe otro tipo de operaciones encargadas de convertir tipos de datos string a un valor numérico.

  • Operadores de incremento y decremento: Cuando queremos aumentar o disminuir el valor numérico de una variable es común realizar lo siguiente: El signo que realiza la adición en la suma se escribe ++ (con dos signos más seguidos) y en el caso la resta se escribe -- (dos signos menos seguidos).

Conversión de texto a número:

Vamos a ver dos métodos: parseInt y parseFloat...
Empezamos con parseInt:
Convierte una cadena de texto en un número entero, siempre que el String comience con un número entero.
Si el texto contiene un número con decimales el resultado será la parte entera de ese número, eliminando los decimales. Si hay espacios en blanco antes o después del número, van a ser ignorados. Si el string comienza con un número y luego le sigue otro texto, esto último es ignorado.

Concatenación:

Para los numeros podemos sumar, restar, multiplicar y dividir.
Para los strings podemos concatenar.
Cuando trabajamos con strings, el operador + tiene una función distinta a la de sumar, y lo que hace es UNIR los dos strings.
Ejemplos:
"Hola" + "Juan" = "HolaJuan";
"Aguante " + "Velez " + "Sarsfield." = "Aguante Velez Sarsfield."
