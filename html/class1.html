Introducción a JavaScript:

¿Qué es JavaScript?

JavaScript es un lenguaje de programación que se usa principalmente para crear sitios web dinámicos. Con este lenguaje, podemos desarrollar tiendas online, buscadores, gestores de contenido, entre otros.
Decimos que JS es un lenguaje interpretado, porque el código que escribimos se puede ejecutar directamente en el navegador, lo cual significa que los archivos HTML con código JavaScript asociado se interpretan inmediatamente.
JavaScript se utiliza para crear la lógica, el "cerebro" de nuestro sitio web, agregando interactividad a las páginas que ya estructuramos con HTML y a las que dimos estilos con CSS.

Ventajas de JS

  • Sencillo: De todos los lenguajes de programación, es de los más sencillos por la simplicidad que ha alcanzado en la forma de escribir su código o sintaxis.
  • Rápido: Tiende a ejecutar las funciones inmediatamente, sin requerir de una compilación o emulador.
  • Añade interactividad: Permite que el usuario pueda interactuar con la información.
  • Ampliamente soportado: Puede ser ejecutado en todos los navegadores más populares y con ello ingresa en todo dispositivo que use internet.
  • Versátil: Permite desarrollar páginas web dinámicas y aplicaciones web.
  • Multiplataforma: Puede ser ejecutado de manera híbrida en cualquier sistema operativo móvil.
  • Fullstack: Permite trabajar tanto del lado cliente como del servidor.

Funciones Nativas

Una función son unos bloques de código que nos permiten ejecutar ciertas instrucciones.

  • Alert: Sirve para mostrar un mensaje en una ventana emergente en el navegador.
    En el archivo JS se escribe de la siguiente manera...
    alert("Mi primer función");
  • Console.log: Sirve para mostrar un mensaje pero desde la consola.
    En el archivo JS se escribe de la siguiente manera...
    console.log("Mensaje por consola")
  • Prompt: Esta otra función nos brinda acceso a una mayor interacción con el usuario ya que podemos pedirle sus datos a través de ella. La respuesta puede ser guardada en una variable.
    En el archivo JS se escribe de la siguiente manera...
    prompt("Ingrese su nombre")
