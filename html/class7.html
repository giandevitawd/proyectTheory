Relación entre arrays y objetos:

Un objeto en JavaScript es un conjunto de datos organizados en pares clave-valor, donde cada valor puede ser un número, una cadena de texto, una función o incluso otro objeto. Es como una caja que contiene diferentes elementos con etiquetas para identificarlos y acceder a ellos. Los objetos son muy útiles para representar cosas con características y comportamientos en un programa.
Aunque ya conocemos los arrays, no viene mal repasar para refrescar la memoria... Un array en JavaScript es como una lista de cosas que puedes guardar en una variable. Puedes poner diferentes tipos de cosas en el array, como números, palabras, objetos e incluso otros arrays. Luego, puedes acceder a cada uno de los elementos del array por su posición en la lista.

Pero, ¿Cómo los relacionamos entre sí?

En JavaScript, un array es un tipo especial de objeto que almacena una colección de elementos indexados numéricamente. Por lo tanto, los arrays también son objetos en JavaScript. La principal diferencia entre un array y un objeto es que en un array los elementos están ordenados y se acceden mediante índices numéricos, mientras que en un objeto los elementos se acceden mediante claves o nombres descriptivos. Aunque ambos comparten algunas similitudes, son tipos de datos diferentes y se utilizan en diferentes situaciones en el desarrollo de aplicaciones web en JavaScript.

Objeto Math:

El objeto Math es una herramienta que proporciona funciones matemáticas útiles en JavaScript. No necesita crear un objeto Math para usarlo, ya que es un objeto global predeterminado en este lenguaje. En el contexto de un sitio web e-commerce suceden muchos cálculos a la vez, por ejemplo suma de precios de productos en un carrito, descuentos, impuestos, etc. Todos estos cálculos pueden llevarse a cabo por medio del objeto math.

Métodos en objetos:

Un método en JavaScript es como una acción que puede hacer un objeto. Por ejemplo, si un objeto es un perro, un método podría ser "ladrar". Los métodos son funciones que están dentro de un objeto y que pueden ser llamados para realizar una tarea específica en ese objeto. Pueden aceptar argumentos, modificar propiedades del objeto y devolver un resultado. Es como si el objeto tuviera habilidades especiales que puede realizar cuando se lo llama para hacerlo.
Un método puede cambiar las propiedades del objeto, calcular valores en función de esas propiedades, realizar acciones en el contexto del objeto, o simplemente devolver un valor de mismo objeto. Los métodos son una parte importante de la programación orientada a objetos en Javascript y se utilizan frecuentemente para organizar el código y la lógica de un programa.
