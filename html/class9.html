.createElement:

createElement es un método en JavaScript que se utiliza para crear un nuevo elemento HTML. Permite crear elementos como div, p, span, entre otros, de forma dinámica desde el código JavaScript.
La sintaxis básica para utilizar createElement es la siguiente:
document.createElement(nuevoElemento);
En donde “nuevoElemento" es el nombre del elemento HTML que deseas crear. Por ejemplo, si deseas crear un nuevo elemento div, puedes hacerlo de la siguiente manera:
const nuevoDiv = document.createElement('div');
Una vez que hayas creado el elemento, puedes manipularlo y agregarlo a tu página web utilizando otros métodos, como appendChild() para añadirlo como hijo de otro elemento existente.
Ahora bien, nuevoDiv, por el momento, es sólo una variable para nuestro programa y no está conectado al DOM o no está insertado en nuestro archivo HTML. Para poder ver si efectivamente el elemento existe y si forma parte del código HTML, podemos utilizar el método “isConnected”. isConnected es una propiedad booleana en JavaScript que se utiliza para verificar si un elemento HTML está conectado o adjunto al árbol del documento.
La propiedad isConnected devuelve true si el elemento está conectado al árbol del documento, lo que significa que está presente en la estructura de la página web y se encuentra disponible para su interacción. Por otro lado, si el elemento no está conectado, la propiedad isConnected devuelve false.
Puedes utilizar la propiedad isConnected para verificar si un elemento ha sido eliminado del árbol del documento o si aún está presente en la página. Esto puede ser útil, por ejemplo, para realizar acciones o ejecutar código dependiendo de si un elemento está visible en el documento o no.

appendChild es un método que se utiliza en JavaScript para agregar un nuevo elemento (nodo) como hijo de otro elemento existente en el DOM (Documento de Objetos del Modelo). Lo importante a tener en cuenta con este método es que el nuevo elemento que se agregue se suma al final de una lista de elementos contenidos en un “padre". Eliminado elementos desde JavaScript:

Así como se pueden crear nuevos nodos o elementos HTML a partir de métodos en JavaScript, también podemos eliminarlos, o en algunos casos, vaciarlos. El método remove() es una función en JavaScript que se utiliza para eliminar un elemento del árbol DOM. Permite eliminar un elemento HTML y todos sus descendientes del documento web. La sintaxis básica del método remove() es la siguiente:
elemento.remove();
Donde elemento es la referencia al elemento que se desea eliminar.
Al llamar al método remove() en un elemento, este se elimina completamente del DOM, lo que implica que ya no será visible en la página web y se eliminarán todos los eventos y datos asociados a él.
Otra manera de “hacer invisibles” elemento del HTML es simplemente mostrándolos vacíos sin que dejen de existir. Para entenderlo mejor, repasemos .innerHTML: Es una propiedad que representa el contenido HTML dentro de un elemento. Permite acceder y modificar el contenido HTML interno de un elemento, incluyendo etiquetas, texto y otros elementos HTML anidados. Al asignar un valor a `innerHTML`, se reemplaza todo el contenido anterior del elemento. Por ejemplo, el contenido de un elemento p es un texto. Lo que podemos hacer con esta propiedad es modificar el texto que contiene ese elemento desde Javascript. En el caso de que queramos eliminar un texto de nuestro sitio podemos utilizar el método .remove() para remover el p pero debemos tener en cuenta que el elemento dejará de existir y si de ser necesario, debe ser creado nuevamente. Otra manera es simplemente completar con un texto vacío, es decir, darle un contenido vacío.
