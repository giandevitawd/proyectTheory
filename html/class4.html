Funciones:

Las funciones nos permiten empaquetar o agrupar líneas de código para poder utilizarlas múltiples veces sin generar repetición inecesaria. También nos permite dinamizar código.
Las funciones se crean utilizando la palabra reservada function, tienen un nombre, llevan un paréntesis y utiliza llaves para poner las líneas de código que pertenecen a la función.
Para ejecutar la función simplemente tenemos que escribir el nombre de la función seguido por ().

Otras maneras de declarar a las funciones:

  • Guardar la función con nombre en una variable: Consiste en crear una variable y asignarle una función. Este método se presta a confusión porque cuando llamamos a la función tenemos que hacerlo por el nombre de la variable y no por su propio nombre. En este ejemplo la llamada es suma().
  • Declarar una función anónima: Consiste en declarar una función sin nombre. A esta función si la tenemos que guardar en una variable para poder llamarla después.
  • Función flecha o Arrow function: A partir de la revisión ES6 de Js se agregó una forma abreviada de declarar funciones anónimas, las arrow function. También las guardamos en variables pero reemplazamos la palabra reservada function por una flecha =>. Es la manera recomendada en la mayoría de los casos.

Scope:

En la programación, según donde declaremos una variable, podemos determinar dónde se va a utilizar dicha variable, esto se conoce como ámbitos de la variable o scope.
Dependiendo del lugar de creación, identificamos a las variables en dos tipos:

  • Variables globales: si una variable se declara fuera de cualquier función o bloque, automáticamente se transforma en variable global.
    Usamos variables globales cuando queremos emplearlas en distintos bloques del programa.
  • Variables locales: cuando definimos una variable dentro de una función o bloque, la identificamos como una variable local, y será accesible sólo dentro de ese espacio. Si quisiéramos utilizarla por fuera, la variable no existirá para el intérprete.
    Usamos variables locales cuando sólo son necesarias para la función o estructura actual.
